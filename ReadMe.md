# Course repository for Data Science course (HA and Bsc(B))

Date Science is a 5 ECTS course at bachelor level (4th semester) for HA (Højre Afsætning, taught in Danish) and BSc(B) (Bachelor of Science in Business, taught in English).

The course repository serves as a central hub for all course-related materials, including lecture notes, assignments, and administrative documents. It is designed to facilitate easy access and organization of course content for instructors.

## Structure

The course repository is structured as follows:

* admin: Contains administrative materials, such as course syllabus and schedule 
   


* lectures: Contains lecture materials, including slides and recordings

## Course tools

The course utilizes the following tools:

   * Programming language: R
   * Packages: tidyverse, ggplot2, dplyr, and other relevant packages for data manipulation and visualization
   * IDE: RStudio (used during exam and tutorials)
   * IDE cloud: Posit Cloud (used for lectures)
   * Version control: Git and GitHub
